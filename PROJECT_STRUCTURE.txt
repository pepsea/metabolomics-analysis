═══════════════════════════════════════════════════════════

├── 🚀 EXECUTION FILES

│
├── 📚 DOCUMENTATION
│   ├── SETUP_COMPLETE.md ⭐ START HERE

│

│   ├── requirements.txt         (Python dependencies)

│
└── 📊 OUTPUT FILES (Generated)
    ├── fetched_papers.json     (Intermediate: Raw fetched papers)

═══════════════════════════════════════════════════════════
QUICK START
═══════════════════════════════════════════════════════════

1. Install dependencies:
   $ pip install -r requirements.txt

2. Set API key:
   $ export ANTHROPIC_API_KEY='your-key'

3. Run:
   $ ./run_literature_radar.sh

4. Check output:
   $ cat literature_radar_output.json

═══════════════════════════════════════════════════════════
CUSTOMIZATION OPTIONS
═══════════════════════════════════════════════════════════

Preset Configs (via generate_config.py):
  • default             - Standard configuration
  • quick               - Fast 2-day scan
  • comprehensive       - Deep 90-day analysis
  • peer_reviewed_only  - Exclude preprints
  • preprints_focus     - Prioritize preprints
  • metabolomics_only   - Metabolomics + Lipidomics
  • ai_focus            - AI/Agents + Bioinformatics
  • weekly              - Weekly reports
  • monthly             - Monthly deep dive
